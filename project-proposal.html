Purpose

My website will be designed around giving information about a game I like to play called Overwatch. I plan to go into details about various things like characters, controls, and game types.

Web Pages

  • Home: This will give a brief description about the game Overwatch.
  • Characters: A page with an image and description of each of the playable characters in the game.
  • Controls: An explanation of each control item such as primary fire or alternate fire, and what button or key to press depending on the system you are playing on.
  • Game Modes: Descriptions of the different game modes that can be played.
  • Unlockables: Descriptions of what you can unlock in the game and how you unlock items.
