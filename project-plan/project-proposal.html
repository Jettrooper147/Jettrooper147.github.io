Home

Purpose

My website will be designed around giving information about a game I like to play called Overwatch. I plan to go into details about various things like characters, controls, and game types.

Web Pages

  • Home: This will give a brief description about the game Overwatch.
  • Characters: A page with an image and description of each of the playable characters in the game.
  • Controls: An explanation of each control item such as primary fire or alternate fire, and what button or key to press depending on the system you are playing on.
  • Game Modes: Descriptions of the different game modes that can be played.
  • Unlockables: Descriptions of what you can unlock in the game and how you unlock items.

Target Audience

This website will be a guide for new Overwatch players or for people considering trying the game. There will be basic information about game modes and characters that new players could find useful.

Persona

  • Name: Tim Johnson
  • Occupation: College student
  • Age: 20
  • Interests: Video games, guitar
  • Motivation: Tim likes playing video games, but has never tried Overwatch. He thinks it looks fun, but there are so many characters to choose from and game modes to play, he doesn't know where to start
  • Environment: As a college student, Tim is very busy. When he isn't in class though he likes to relax by playing some video games or by picking up his guitar.
